AMD OSX86 Post Install

  • 日 13 8月 2017
  • osx

  • #osx, #osx86, #amd

I migrate from and Intel system, here are my steps.

  1. Prepare usb disk with TransMac
  2. Boot USB Disk, BIOS set to UEFI+Legacy mode.
  3. Launch terminal from installer

sh /Volumes/SierraAMD/PostInstall/post.sh The script will ask for your OSX volume, but volume name may not contains spaces. If your OSX volume name contains spaces, make soft link first

cd /Volumes/ ln -s OS\ X OSX

  1. Reboot, first time boot with USB.
  2. Install CLOVER
  3. Make kernel caches

sudo kextcache -system-prelinked-kernel sudo kextcache -system-caches Then you should be able to boot with clover

  1. If you have monitor color blinking issue, install ResXtream and set 8-bits color
  2. If your system clock runs too fast/slow, adjust busratio in clover boot arg
  3. I found there is a process assistantd keeps writing to my disk, stopping Siri does not help. Thus I disable the service directly

``` sudo vim /System/Library/LaunchAgents/com.apple.assistantd.plist

#change RunAtLoad to false and reboot ```

  1. Other issue: playing video with VLC got crappy audio output. Use mpv or mplayer
